## 소개
슬롯머신을 통해 랜덤한 메뉴를 고를 수 있도록 도와주는 안드로이드 앱입니다.
<br/><br/>

### | 화면 구성
- 메인 화면 : (+) 버튼을 통해 메뉴 목록으로 이동하거나, (START)버튼으로 슬롯머신을 돌릴 수 있습니다.

- 메뉴 목록 회면 : (+) 버튼을 통해 메뉴를 추가하거나, 검색으로 메뉴를 조회할 수 있습니다.
